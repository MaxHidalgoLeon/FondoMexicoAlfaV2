# Fondo Mexico Strategy Research

This project is a skeleton for a Mexico-focused long-biased industrial strategy that blends macro, fundamental, and technical signals with institutional risk controls and FX management.

## Project structure

- `src/` - reusable research modules for data, features, signals, portfolio construction, backtesting, and risk.
- `data/` - placeholder for raw data and external files.
- `notebooks/` - exploratory research notebooks.
- `reports/` - output summaries and visualizations.
- `tests/` - lightweight unit tests for the pipeline.

## Next steps

1. Replace the mock data generator in `src/data_loader.py` with real Mexican equity, FIBRA, logistics, macro, and FX data.
2. Use the example pipeline in `src/pipeline.py` to validate the universe, signals, and backtest.
